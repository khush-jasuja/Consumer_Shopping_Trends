# 📊 Consumer Shopping Trends 2026 – Power BI Dashboard

## 📌 Overview

This project presents a comprehensive analysis of consumer shopping behavior using a dataset of **11,789 records** and **25 features**. The dataset captures demographic, behavioral, and psychographic factors influencing how consumers shop across online and offline channels.

The Power BI dashboard built from this dataset provides actionable insights into purchasing patterns, customer preferences, and digital adoption trends.

---

## 📂 Dataset Summary

* **Total Records:** 11,789
* **Total Features:** 25
* **Data Type:** Mixed (Numerical + Categorical)

### 🧾 Key Feature Categories

#### 1. Demographics

* `age`
* `gender`
* `monthly_income`
* `city_tier`

#### 2. Digital Behavior

* `daily_internet_hours`
* `smartphone_usage_years`
* `social_media_hours`
* `tech_savvy_score`

#### 3. Online Shopping Behavior

* `monthly_online_orders`
* `avg_online_spend`
* `online_payment_trust_score`
* `product_availability_online`

#### 4. Offline Shopping Behavior

* `monthly_store_visits`
* `avg_store_spend`
* `need_touch_feel_score`

#### 5. Consumer Psychology

* `impulse_buying_score`
* `brand_loyalty_score`
* `discount_sensitivity`
* `environmental_awareness`

#### 6. Logistics & Experience

* `avg_delivery_days`
* `delivery_fee_sensitivity`
* `free_return_importance`
* `return_frequency`

#### 7. Lifestyle Factors

* `time_pressure_level`

#### 8. Target Variable

* `shopping_preference` (Online / Store / Hybrid)

---

## 📊 Dashboard Objectives

The Power BI dashboard aims to:

* Identify key drivers of **shopping preferences** (Online vs Store vs Hybrid)
* Analyze the impact of **income and demographics** on spending habits
* Understand **digital adoption trends** and their influence on purchasing behavior
* Evaluate **customer sensitivity** to discounts, delivery fees, and returns
* Segment customers based on **behavioral and psychological traits**
